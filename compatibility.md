# Compatibility with Bootstrap Versions

## For Bootstrap 5.x

```bash
npm install bootpag
```
Or use a CDN:

**CDN jsDelivr**
```html
<script src="https://cdn.jsdelivr.net/npm/bootpag/dist/jquery.bootpag.min.js"></script>
```

**CDN unpkg**
```html
<script src="https://unpkg.com/bootpag/dist/jquery.bootpag.min.js"></script>
```
